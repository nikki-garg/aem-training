Day 2 : Operator concept

Comparison Operator

Age is 18 or older: ${age >= 18}

Logical Operators

Can borrow books: ${(age >= 18) && hasLibraryCard}

Has library access: ${isStudent || hasLibraryCard}

Conditional Operators

Eligibility: ${age >= 18 ? 'Adult' : 'Minor'}

Student Discount: ${isStudent ? 'Yes' : 'No'}

Grouping with Parentheses

Special Access: ${ (isStudent && (age < 25)) || hasLibraryCard }

Welcome Adult Student!
